# Cara pakai
Update package
```
$ pkg update && pkg upgrade 
```

Install modul
```
$ pkg install git

$ pkg install python

$ pip install requests
```

Install tools
```
$ git clone https://github.com/omKeyx/IPLocation

$ cd IPLocation

$ python IPLocation.py
```
